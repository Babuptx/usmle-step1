# USMLE Step 1 Master Bank

A comprehensive, interactive, and free-to-use board preparation platform designed to help medical students master the high-yield concepts required for the USMLE Step 1 examination. 

This custom-built engine features **560 elite-level questions** divided into 14 progressive blocks, bridging the gap between foundational physiology and complex, multi-system clinical pathology.

## 🎯 Intended Audience
This platform is intended for **medical students** globally who are currently studying for the United States Medical Licensing Examination (USMLE) Step 1, or the COMLEX-USA Level 1. It serves as a rigorous, supplementary tool for active recall, clinical reasoning, and testing stamina. It is provided entirely free of charge to support open-access medical education.

## ✨ Key Features
* **Massive Question Bank:** 560 unique, multi-order clinical vignettes.
* **Progressive Difficulty:** 14 distinct blocks (7 Foundations, 7 Clinical Integration) mapped to a visual green-to-red difficulty spectrum.
* **In-Depth Rationales:** Detailed explanations for correct answers and explicit breakdowns of why every incorrect option is wrong, including the core "Educational Objective" for every question.
* **Interactive UI:** Mimics official board testing software to reduce cognitive load on exam day. Features include question flagging, an interactive navigation grid, and visual tags for Discipline and System.
* **Persistent Scoreboard:** Tracks your overall average and block-specific scores across study sessions using local browser storage.
* **Quality of Life Tools:** Includes a Dark Mode toggle for late-night studying, a Question Shuffle feature to prevent pattern memorization, and a quick-reference Standard Laboratory Values modal.
